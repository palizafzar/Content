نصب مرحله به مرحله نرم افزارهای لازم به صورت زيرمي باشد:

1. نصب نرم افزار SQL Server2008R2 (لینک نصب SQL)

2. نصب نرم افزار حضور و غیاب: با اجرای ستاپ برنامه InOutControlManagement_Setup.exe در حالت Run As Administrator نصب را تا انتها ادامه دهید.

با نصب ستاپ برنامه، برنامه پالیز استودیو، نرم افزار حضور و غیاب و همچنین برنامه جانبی Database Installation Wizard(جهت اتصال بانكهاي اطلاعاتي نرم افزارها به SQL سرور وبه روز رساني بانكها SQL) نيز نصب خواهد شد.

3. برای برنامه جامع، قفل دانگل USB را به یکی از پورتهای کامپیوتر متصل نمایید

4. اکنون لايسنس نرم افزار InOutControlManagement.LS.lic را در مسير نصب برنامه در شاخه System کپی نمایید. در نسخه لایت (رایگان) از لایسنس موجود در شاخه

tools\standard LIC

استفاده نمایید. و در نسخه جامع از لایسنس موجود در CD کوچک همراه قفل دانگل USB خود استفاده نمایید. مسير پيش فرض نصب برنامه به صورت زير مي باشد:

C:/Program Files/ Paliz Afzar Company/ Paliz Afzar Studio

نصب در اینجا تمام است و باید تنظیمات اولیه برنامه را انجام دهید.

5. درصورت داشتن بک آپ از برنامه، پس از نصب برنامه باید دیتابیس مورد نظر را به برنامه متصل نمایید تا بتوانید از اطلاعات قبل استفاده نمایید و نیاز به تعریف مجدد نداشته باشید. برای این منظور از برنامه دیتا بیس اینستالیشن ویزارد استفاده می کنیم.

Database Installation Wizard

با استفاده از این برنامه می توانید دیتابیس های موجود را به برنامه متصل نمایید و درصورت نیاز به بروزرسانی آنها را بروزرسانی نمایید. همچنین برای گرفتن بک آپ و بازگرداندن اطلاعات بک آپ گرفته شده به برنامه از این برنامه استفاده می گردد.

1- از گزينه Start ویندوز و شاخه PalizAfzarproducts، گزينه DatabaseInstallationWizard را انتخاب مي نماییم. کلید Next را بزنید.

2- در صفحه اخیر به دو گزينه توجه داريم نام سرور(Server Name) و اتصال ديتابيس:

3- اگر نام سرور SQL نامشخص باشد كلمه Not Specified در کادرServer Name قابل مشاهده است.

4- درصورت عدم اتصال پايگاه داده به سرور مورد نظر، در كادرDatabase Connection Statue عبارت Connection Failed درج شده است. بنابراين براي انتخاب نام SQL Server مورد نظر و برقراري ارتباط با آن كليد Property را مي‌زنيم.

5- در پنجره Property بايد نام سروري را كه مي‌خواهيم با آن ارتباط برقرار كنيم وارد نماييم. همچنين مي‌توان از طريق دکمه سه نقطه لیست سرورهای SQL (پايگاه داده) در دسترس را مشاهده نموده و سرور مورد نظر را انتخاب (Select) نمود

سپس كليد Connection test را جهت تست ارتباط می زنیم. پیغامی با عنوان (ارتباط برقرار شد) Connection Ok را مشاهده خواهيم نمود و بعد دکمه Apply مي‌زنيم.. پس از آن كليد Close را مي‌زنيم.

6- اکنون نام سرورSQL انتخابي ما قابل مشاهد خواهد بود و كادرDatabase Connection Statue عبارت Done را نمايش مي‌دهد. اكنون دکمه Next را مي‌زنيم.

7- در این مرحله بايد ديتابيسهاي مورد نياز را به پايگاه داده SQL متصل نمود. به اين عمل Attach گفته مي‌شود. براي اتصال همه ديتابيسهاي مورد نظر مي‌توان آنها را انتخاب نمود و دريك مرحله Attach نمود. ابتدا گزينه Attach Databases و سپس ديتابيسهاي In Out Control Databases و Paliz Afzar Studio Databases را انتخاب نموده و كليد Next را مي‌زنيم.

8- درصفحه كنوني دکمه Next را می‌زنیم. درصورتي كه آدرس ديتابيس مشخص نباشد ويا بخواهيم ديتابيس مورد نظر خود را از شاخه اي ديگر فراخواني كنيم.

از كليد Browse استفاده مي‌كنيم. و مسير و فايل Data Base را انتخاب مي‌كنيم و در پنجره بعد كليد Finish را مي‌زنيم.

9- پس ازAttach نمودن ديتابيسها، آنها را بروزرساني (update) می کنیم. برای این منظور از طريق برنامه Database Installation Wizard گزينه‌ Update Databases وديتابيس Paliz Afzar Studio را انتخاب نموده كليد Next را مي‌زنيم.

10- حال جهت فراخواني فايل Update، كليد Browse را مي‌زنيم. از شاخه Databases Update فايل Palizafzar Studio-hf.pak را انتخاب نموده كليد Open را مي‌زنيم. Update نمودن In Out Control Management نيز همين روش را انجام مي‌دهيم. فقط فايل Update، InOutControlManagement-HFxx.pak را بايد انتخاب نمود.

پس از نصب برنامه برای فارسی شدن زبان برنامه عملیات زیر را انجام دهید.

به کنترل پنل ویندوز رفته و تنظیمات زیر را انجام داده ذخیره نمایید

  • Control panel -> Region -> Format = persian
  • Control panel -> Region -> Administrative ->Change System Locale= persian(Iran)
  • Control panel -> Region ->additional Setting -> Number -> Decimal symbol = " . "
  • Control panel -> Region ->additional Setting -> Date -> Short date = MM/DD/YYYY
  • Control panel -> Region ->additional Setting -> Date -> Calendar type =( میلادی(انگلیسی)

اجرای برنامه

روی گزینه Paliz Afzar Studio در صفحه دسکتاپ كلیك نمایید. استودیوی كنترل محصولات پالیزافزار شامل دو پنجره گشوده می شود.

مدیر اجازه دسترسی: در اینجا نمایش نام محصول، ویرایش، نگارش و وضعیت دسترسی نرم افزار را داریم.

  • نام محصول: نام نرم افزار قابل اجرا تحت استودیو را نمایش می‌دهد.
  • ویرایش محصول: سطح كاربرد و یا امكانات مورد استفاده از نرم افزار را مشخص می‌نماید.
  • شماره نگارش: هر محصول مورد استفاده دارای شماره نگارش می‌باشد.
    • علت نمایش پیام "فایل اجازه دسترسی پیدا نشد" و یا " اجازه دسترسی وجود ندارد" در وضعیت دسترسی
  • وضعیت دسترسی: دسترسی به نرم افزار توسط استودیوی پالیزافزار كنترل می‌گردد دراینجا نسبت به عدم دسترسی و یا دسترسی محدود و یا دسترسی كامل پیغام مناسب در قسمت وضعیت دسترسی درج می‌گردد.
  • درصورتیكه وضعیت دسترسی جمله "فایل اجازه دسترسی پیدا نشد" را نمایش دهد، باید فایل لایسنس نرم افزار درشاخه System، محل نصب برنامه كپی شود. مسیرپیش فرض نصب برنامه:
  • C:\Program Files\Paliz Afzar Company\Paliz Afzar Studio\System
  • اشكال در عدم تطابق فایل لایسنس با قفل نرم افزار و یا عدم خواندن قفل توسط Windows می‌باشد.
    • با كلیك راست ماوس روی "مدیریت حضور و غیاب پالیز" منویی شامل دو گزینه بازخوانی و مشخصات ظاهر می‌گردد.
  • بازخوانی: برای بازخوانی و كنترل مجدد لایسنس توسط استودیوی پالیزافزار می‌توان این گزینه را انتخاب نمود.
    • مشاهده مشخصات و تعداد نفرات لایسنس
  • مشخصات: این گزینه اطلاعات كامل لایسنس موجود شامل شماره سریال قفل، ویرایش محصول، تعداد پرسنل قابل تعریف، سقف زمان استفاده از قفل و... را نمایش می دهد.
  • تنظیمات پایگاه داده و اتصال (اتچ) آن به SQL

مدیر پایگاه داده: هنگام نصب برنامه، دیتابیس نرم افزارحضوروغیاب و استودیوی پالیزافزار به پایگاه داده SQL متصل (Attach) شده است. این امكان وجود دارد كه كه دیتابیسهای نرم افزاربه پایگاه داده های مختلف متصل شوند. دراینجا می‌خواهیم مشخص كنیم نرم افزار حضوروغیاب با كدام پایگاه داده SQL ارتباط داشته باشد. می‌توانیم ارتباط برنامه را با SQL یك كامپیوتردیگر برقرار نماییم. یعنی از دیتابیس كامپیوتر دیگر استفاده نماییم.

برای این منظوردر قسمت مدیر پایگاه داده بر روی مدیریت حضور و غیاب پالیزافزار، كلید راست ماوس را زده گزینه مشخصات را انتخاب می‌كنیم. پنجره مشخصات Property گشوده می‌شود.

نام سرور SQL مورد نظر را در كادر نام سرور (Serv erName) وارد می نماییم. و یا از كلید برای جستجو و انتخاب سرور، استفاده و كلید اعمال را می زنیم. اگر پایگاه داده انتخاب شده جهت اتصال، در كامپیوتر دیگری باشد طبق مجوزهای داده شده در هنگام نصب SQL می توان از مجوز ویندوز یا پایگاه داده استفاده کرد شكل(11)

با زدن دكمه تست ارتباط، در صورت برقراری ارتباط پیام ارتباط برقرار است روی صفحه ظاهر می‌شود. شكل(12)

پس از اطمینان از برقراری ارتباط میان نرم افزار و پایگاه داده انتخاب شده (SQL) كلید اعمال را می زنیم تا تغییرات ذخیره شود

سپس کلید بستن را می‌زنیم شكل (11).

به صفحه اصلی مدیریت كنترل استودیوی پالیز باز می گردیم. وضعیت ارتباط در مدیر پایگاه داده عبارت ارتباط برقرار است را نمایش می‌دهد. شكل (10).

اكنون می‌توانیم وارد برنامه شویم.

در بالای پنجره استودیوی كنترل محصولات پالیزافزار چهار گزینه وجود دارد

شروع: برای ورود به برنامه ازاین گزینه مدیریت حضوروغیاب پالیز را انتخاب می‌كنیم.شكل(13)

  1. نمایش: با انتخاب گزینه هدایتگر لیست گزینه های نرم افزار قابل مشاهده می‌باشد.
  2. كمک: نگارش استودیوی كنترل محصولات پالیزافزار را نمایش می‌دهد.شكل(14).

نام کاربر و رمز پیش فرض برنامه لایت و یا جامع

با انتخاب مدیریت حضوروغیاب پالیز از گزینه شروع وارد برنامه می‌شویم. نام كاربرپیش فرض عدد یک و كلمه عبور پیش فرض نیز عدد یک می‌باشد شكل(15)

تعریف کاربر با سطح دسترسی دلخواه (لینک تعریف کاربر)

امكان تعریف كاربران مختلف با سطوح دسترسی متفاوت از منوی مدیریت کاربران وجود دارد. برای تعریف کاربران و تعیین سطح دسترسی آنها، به برنامه وارد شده و از منو مدیریت حضوروغیاب، مدیریت کاربران را انتخاب کنید شكل(16)

درپایین صفحه اخیر دو گزینه كاربران وسطوح دسترسی وجود دارد. (شکل 17)

سطوح دسترسی: در اینجا امكان تعریف سطوح دسترسی مختلف با نام وكد وجود دارد.

  • ابتدا كد و نام مورد نظر خود جهت دسترسی به بخشهای دلخواه را تعریف کنید. سپس از پنجره سمت چپ گزینه‌های مورد نظر خود را علامت زده و در انتها كلید ذخیره را می‌زنیم.

ابتدا كد و نام مورد نظر خود جهت دسترسی به بخشهای دلخواه را تعریف کنید. سپس از پنجره سمت چپ گزینه‌های مورد نظر خود را علامت زده و در انتها كلید ذخیره را می‌زنیم.
