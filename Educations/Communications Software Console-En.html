This console software is used to communicate with the attendance and access control devices to set up all their sections. However, keep in mind that this software will communicate with the relevant software and service of the attendance and access control devices.

For establishing communication between the console software and the service software, the IP address and port number of the network must be configured. To do this, press the change key, make the desired changes or settings, and then save them. Pay attention to the communication status option to check if the connection is established.

If the connection is successful, the device name will be displayed in the list of available devices. The name and communication specifications of the attendance and access control device are defined in the service software.

Time Settings

This section allows setting, receiving, and sending time-related information such as date, time, display format, and calendar type.

Main Software Settings

Here, settings like language selection and database configuration are available.

Card Reader Settings

Settings for internal and external card readers, including enabling or disabling, and specifying card types.

Device Security Settings

Configuration options for device security, including password settings and error detection.

Card Data Retrieval

Options for receiving card-related data, specifying criteria such as date, time, and card type.

Card Data Transmission

Configuration settings for sending card data, including card identification, validation, and transmission interval.

These settings provide comprehensive control over the attendance and access control devices, ensuring efficient and secure operation.
